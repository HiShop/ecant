# ECAnt 电商蚂蚁
一个电商信息爬虫工具，用于爬取主流电商平台（如京东、淘宝）的分类、品牌、商品等基础信息，形成统一格式的大数据库。

# 数据使用场景
1. 电商类软件平台的测试数据样本。
2. 特定经营类目的数据导入，可模拟垂直细分领域的商户生产数据。
3. 阶梯规模的性能测试数据导入。
4. 基于大数据的商品定价辅助决策。

# 开放能力
1. 统一开发的数据格式，分类、品牌、商品基本信息（规格、价格）、详情资料、缩略图、高清（大）图。
2. 开放透明的数据访问接口，RESTful API风格。

# 路线图
1. 基础数据的爬取。
2. API接口实现。
3. 更多主流电商平台的支持。

# 技术栈
编程语言：Python
数据存储：MongoDB / MySQL
